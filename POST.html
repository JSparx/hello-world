During the last week of October there was likely a new version of ADOBE Reader and ADOBE Standard installed on your computer. This change has created some confusion around how to use the text recognition feature many have become accustomed to using. Please refer to the steps below for instructions on how to use this feature in the NEW version of ADOBE Standard. Please note there are two different options to utilize the text recognition feature.

    • Option 2

How to use text recognition in new
version of ADOBE ACROBAT STANDARD

1. Export the document from EDM (Skip this step if your docs open directly in ADOBE ACROBAT)

2. Locate the document in your Downloads folder (Skip this step if your docs open directly in ADOBE ACROBAT)

3. Open the document (the document should open in Adobe Acrobat Standard if you have this application)

4. Select the ‘Tools’ panel within ADOBE and select ‘Export PDF’

5. Select ‘Microsoft Word’ then ‘Export’

6. ‘Save’ the word document to a location on your computer

7. After some processing, the document will open up in a word document where you can now copy and paste text as needed

Option 2

ALTERNATE PATH STARTING FROM STEP 3

4a. After exporting and opening your document – select the ‘Enhance Scans’ option

5a. Next select ‘Recognize Text’ > In this file

6a. Then Select ‘Recognize Text again. Processing will occur within ADOBE and you should now be able to copy/paste text directly from the document
